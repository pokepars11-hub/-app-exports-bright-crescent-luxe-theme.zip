BRIGHT CRESCENT LUXE - HOSTINGER WORDPRESS INSTALL GUIDE

1) Upload Theme
- Login to WordPress Admin on Hostinger
- Go to Appearance > Themes > Add New > Upload Theme
- Upload: bright-crescent-luxe.zip
- Click Install and Activate

2) Install Required Plugin
- Install and activate Elementor (free)

3) Import Demo Content (optional but recommended)
- Go to Tools > Import > WordPress
- Install Importer if prompted
- Import file: demo-content/bright-crescent-demo.xml

4) Set Home and Blog Pages
- Go to Settings > Reading
- Choose "A static page"
- Homepage: Home
- Posts page: News (optional)

5) Create and Assign Menus
- Go to Appearance > Menus
- Create Primary Menu with pages:
  Home, News, Appliances Products, Crystal Products, Appliances Spare Parts, Contact Us
- Assign to "Primary Menu"
- Create optional footer menu and assign to "Footer Menu"

6) Edit Sections Individually
- Header/Footer details: Appearance > Customize
- Hero/About/CTA/Contact details: Appearance > Customize
- News/Products items: Dashboard > News / Products
- Pages and layouts with drag-and-drop: Edit page with Elementor

7) Contact Form Behavior (as requested)
- Form saves inquiries in Dashboard > Contact Inquiries
- Form also sends email notification to WordPress Admin Email
- Update admin email from Settings > General

8) Catalog Download Button (Spare Parts page)
- Edit "Appliances Spare Parts" page with Elementor
- Select Download button and set your final catalog PDF URL

Elementor Templates:
- Import file from: elementor-templates/home-sections.json
- In Elementor: Templates > Saved Templates > Import Templates
